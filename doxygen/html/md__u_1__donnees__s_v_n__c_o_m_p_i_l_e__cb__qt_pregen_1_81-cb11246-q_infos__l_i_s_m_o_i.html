Extension pour compiler des applications Qt avec 'Code::Blocks'

1-vue d'ensemble 

C'est un plugin pour compiler des applications Qt4 et Qt5 à l'aide de
'Code::Blocks' de manière très semblable que les projets 'wxWidgets.
Pour l'instant il ne fonctionne que sous 'Win32'.

2- Caractéristiques 

Il n'utilise pas les 'Makefile.

Il permet de générer les fichiers complémentaires au projet par
les utilitaires Qt 'moc(.exe)', 'uic(.exe)', 'rcc(.exe)' dans un répertoire
unique du projet actif :
    'qtprebuild/targetname'

1- il détecte les méta-objects 'Q_OBJECT' and 'QGADGETS' dans les fichiers
   *.h, *.hpp, *.cpp et génère  des fichiers 'moc_filename.cpp'

2- il détecte dans les fichiers '*.cpp', les fichiers inclus à la fin par
    '#include "moc_xxxx.cpp' ou '#include "yyyyyy.moc'
    pour marquer ces fichiers inclus en 'no compile' et 'no link',

3- il détecte les fichiers de formes '*.ui' pour générer des fichiers
    'ui_filename.h'

4- il détecte les fichiers de ressources '*.qrc' pour générer des fichiers
    'qrc_filename.cpp'
